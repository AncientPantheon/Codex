# @ancientpantheon/codex-core

Chain-agnostic Codex substrate: the CK-wrapping vault + encryption model, the `CodexAdapter` storage interface, the `CodexSnapshot` data model, the `ForeignChainAdapter` registry seam, the `foreignKeys` keyring model, and the headless resolver factory. No React, no Ouronet, no Pact.

## Status

Version `0.0.1` — an internal workspace package, never published to npm (the consumer-facing surface is `@ancientpantheon/codex`, which bundles this module). Currently an empty, buildable skeleton.
