# @ancientpantheon/codex-ouronet

The Ouronet-native Codex module (lifted from stoa-js): the Kadena/Pact signer, the double-Apollo `ICodexIdentity`, CodexPrime, CodexGuard, seed derivation, the zbom Pact editor, and `ouronet-ns.CODEX` registration.

## Status

Version `0.0.1` — an internal workspace package, never published to npm (the consumer-facing surface is `@ancientpantheon/codex`, which bundles this module). Currently an empty, buildable skeleton.
